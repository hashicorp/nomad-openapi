## Notes for Large OpenAPI documents
If the OpenAPI document is large, imports in nomad_client.apis and nomad_client.models may fail with a
RecursionError indicating the maximum recursion limit has been exceeded. In that case, there are a couple of solutions:

Solution 1:
Use specific imports for apis and models like:
- `from nomad_client.api.default_api import DefaultApi`
- `from nomad_client.model.pet import Pet`

Solution 2:
Before importing the package, adjust the maximum recursion limit as shown below:
```
import sys
sys.setrecursionlimit(1500)
import nomad_client
from nomad_client.apis import *
from nomad_client.models import *
```
